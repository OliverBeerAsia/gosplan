# Release Notes v1.5.0

Date: 2026-02-16

## Highlights

- Building art now mirrors classic SimCity/Transport Tycoon prints: stronger silhouettes, rooftop machinery, and a factory catwalk accent to show industrial scale.
- District-specific glyphs decorate each style before tint modifiers, keeping the Soviet palette but adding unique architectural flourishes per district.
- Release documentation set (implementation notes, multi-agent review, test report, bug check) tracks the new visuals and validation suite.
